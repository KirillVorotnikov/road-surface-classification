# Dataset Guide — Структура данных

Регламент для команды датасета по организации данных.

---

## Структура папок

```
data/raw/
├── asphalt/
│   ├── sample_001.wav
│   ├── sample_002.wav
│   └── ...
├── concrete/
│   ├── sample_001.wav
│   └── ...
├── gravel/
│   ├── sample_001.wav
│   └── ...
├── dirt/
│   ├── sample_001.wav
│   └── ...
└── snow/
    ├── sample_001.wav
    └── ...
```

**Требования:**
- Название папки = класс покрытия (строго на английском)
- Все файлы в формате WAV
- Имена файлов: `sample_{номер}.wav` (например, `sample_001.wav`)

---

## Классы дорожного покрытия

| Класс | Описание |
|-------|----------|
| `asphalt` | Асфальт |
| `concrete` | Бетон |
| `gravel` | Гравий |
| `dirt` | Грунт |
| `snow` | Снег |

---

## Требования к файлам

### Формат
- **Расширение:** `.wav`
- **Кодек:** PCM (без сжатия)
- **Каналы:** Моно или стерео

### Длительность
- Длительность настраивается в конфиге обучения
- Файлы могут быть любой длительности (обрезка/дополнение применяется автоматически)

### Именование
```
sample_001.wav
sample_002.wav
sample_010.wav
```

---

## Загрузка данных

### 1. Разместить файлы

```bash
# Пример для Windows
mkdir data\raw\asphalt
copy C:\recordings\*.wav data\raw\asphalt\
```

### 2. Добавить в DVC

```bash
dvc add data/raw
dvc push
git add data/raw.dvc .gitignore
git commit -m "data: add new samples"
```

Подробно: [DVC_GUIDE.md](DVC_GUIDE.md)

---

## Контроль качества

Перед загрузкой проверить:
- [ ] Файлы в формате WAV
- [ ] Папки названы корректно (asphalt, concrete, gravel, dirt, snow)
- [ ] Имена файлов: `sample_XXX.wav`
- [ ] Файлы не пустые (открываются в плеере)

---

## Обработка данных

Команда ML применяет при обучении:
- Ресемплинг до 16000 Гц
- Конвертация в моно
- Нормализация громкости
- Извлечение Mel-спектрограмм
- Аугментации (шум, питч, time stretch)

Обработанные данные сохраняются в `data/processed/` автоматически и не требуют ручного управления.
